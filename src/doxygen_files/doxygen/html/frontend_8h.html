◆ init_windows()

Windows_t init_windows ( )

Initializes the game windows for the GUI.

Returns
Windows_t A structure containing the initialized game windows.
Note
The function creates four windows: screen, preview, stats, and controls. It sets the color pairs and background color for each window. It also draws boxes around each window.
